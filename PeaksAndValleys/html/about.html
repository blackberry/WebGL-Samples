This WebGL sample's primary intent is to provide a number of WebGL concepts working in one place. This is not intended to be a market-ready application for end consumers. While it can be fun to run around the countryside, this is primarily a WebGL learning resource. Some key concepts include:

  • Image As A Heightmap
  • Endless Terrain
  • Colour Shaders
  • Ambient And Directional Lighting
  • Texture Shaders / Seamless Skybox
  • Multitouch Controls
